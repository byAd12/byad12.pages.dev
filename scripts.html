Scripts

A continuación podrás ver mis scripts públicos.

'parrot.sh' es mi archivo de configuración esencial para Parrot Security OS (basado en Linux).
'proxmox.sh' sirve como menú de administración completo para mi clúster de Proxmox (Chema Hosting).

Comandos para instalar y ejecutar los scripts de forma rápida:

parrot.sh: wget https://byad12.pages.dev/parrot.sh -O parrot.sh; chmod +x ./parrot.sh; ./parrot.sh.
proxmox.sh: wget https://byad12.pages.dev/proxmox.sh -O proxmox.sh; chmod +x ./proxmox.sh; ./proxmox.sh.

Los siguientes botones redirigen al código fuente de cada script:

parrot.sh ↗ proxmox.sh ↗
